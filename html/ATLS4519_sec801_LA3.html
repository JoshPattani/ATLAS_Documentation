Task 1: Create A Basic Slicer for 3D Printing

Creating Geometry

A closed cylinder was created in order to test my basic slicer for 3D printing. This was done in Grasshopper by creating a circle, extruding it, and capping the holes.

Creating the Infill Lines

The cylinder was then sliced into contours along the Z-axis. The bounding geometry of the contours were then used to create lines that would be used as the infill lines.


The series of lines were scaled to be larger than the contours before performing the boolean intersect & join operations.


Once the test polyline was successful, the height of the cylinder was increased to 3 cm to create the layers for slicing.


The infill curves were then discretized into points to create the infill line segments.

Task 2: Create a G-Code Generator in Python/GH

Creating the Gcode

The Gcode was created by using the GHPython script component in Grasshopper. The Gcode was then exported as a text file and saved as a .gcode file.

EXTRA CREDIT: Advanced Slicing and 3D Printing

I decided to try and create a more advanced slicer by creating a variable for infill density. I used the same process as above to create the infill lines and Gcode, added percentage variable components within my Grasshopper algorithm.
